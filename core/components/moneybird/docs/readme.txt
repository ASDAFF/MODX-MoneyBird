--------------------
Extra: MoneyBird
--------------------
Version: 1.0.0

With the MoneyBird package for MODX you're able to create invoice listings (per contact) from your MoneyBird account. After setup you should open the package from the components menu and authorize yourself with username and password or with OAuth. The first time, or as long you didn't setup that you will be promped for it.

See documentation for more details;
http://rtfm.modx.com/display/ADDON/MoneyBird
